# python-calculator-basic
This is my first Python project uploaded on GitHub.
A simple command-line calculator built using Python.

## Features
- Performs +, -, *, /, %, ^
- Continuous execution using loop
- Handles division by zero
- User-friendly input

## Sample Run
Enter first number: 5  
Enter second number: 2  
Enter operator: ^  
Result: 25  

## How to Run
1. Download the file
2. Run: python calc.py
